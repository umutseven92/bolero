# Bolero

## Features

* Orthogonal movement
* Box2D physics
* Load map from Tiled
    * Background layer
    * Foreground layer
    * Collision layer
    * Spawn layer
    * Interaction layer
* NPCs
    * NPC spawn points
    * Talk to NPCs
* Spritesheet animation
* 2D Lighting
* Day / night cycle
