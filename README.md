# Bolero

## Features

* Orthogonal movement
* Box2D physics
* Load map from Tiled
    * Background layer
    * Foreground layer
    * Collision layer
    * Spawn layer
    * Interaction layer
* NPCs
    * NPC spawn points
    * Talk to NPCs
    * Simple schedule
* Sprite sheet animation
* 2D Lighting
* Day / night cycle
* Real time map updates during development
* Read config from file
